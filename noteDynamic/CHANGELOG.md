# Changelog

All notable changes to the QuickNote project will be documented in this file.

## [Unreleased]

### Added

- Initial project setup with React Native 0.78 and TypeScript
- Basic folder structure following Clean Architecture principles
- Firebase configuration files
- Navigation structure with React Navigation v7
- Basic UI components (Button, Input, Card)
- Authentication screens (Login, Register)
- useAuth hook for Firebase authentication
- Zustand store for global state management

- Basic environment variable setup for API keys
