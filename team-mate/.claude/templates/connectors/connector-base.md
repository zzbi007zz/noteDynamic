# Connector Base Template

**Type:** Connector
**Purpose:** Base template for external service integrations

---

## Context

[Service name, authentication method, access level]

## Task

Connect to [service] and perform [action]

## Requirements

1. Use OAuth or secure authentication
2. Follow service API limits
3. Handle errors gracefully
4. Return structured data

## Output Format

```

### Service
[Service name]

### Action
[What was done]

### Data
[Returned data]

### Errors
[Any errors encountered]
```
